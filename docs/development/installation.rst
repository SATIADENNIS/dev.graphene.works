Installation
========================

.. Tip::
   - We recommend building on Ubuntu 16.04 LTS (64-bit)
   - Graphene requires a 64-bit operating system to build, and will not build on a 32-bit OS.
   - Graphene now supports Ubuntu 18.04 LTS
   - Graphene now supports OpenSSL 1.1.0
   - See Graphene :ref:`system requirements <system-requirements-node>`

CLI Tools
-------------------------
Graphene-Core has prepared three CLI wallet tools. You can find the latest cli tool files in `Graphene-Core Release <https://github.com/graphene-blockchain/graphene-core/releases>`_ page.


* Graphene-Core-x.x.xxxxxx-Linux-cli-tools.tar.gz
* Graphene-Core-x.x.xxxxxx-macOS-cli-tools.tar.gz
* Graphene-Core-x.x.xxxxxx-Windows-x64-cli-tools.zip

  - :ref:`How to install CLI tool on Windows x64 <cli-tool>`
